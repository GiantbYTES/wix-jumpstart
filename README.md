# Wix Jumpstart Assignments

Two foundational web development assignments focusing on HTML and CSS fundamentals.

## HTML Assignment
A semantic CV/resume webpage demonstrating proper HTML structure and semantic elements.

**Features:**
- Semantic HTML5 structure
- Profile information with contact details
- Work experience timeline
- Education and skills sections
- Proper use of headings, lists, links, and tables

**File:** `HTML-Assignment/index.html`

## CSS Assignment
A Google-inspired home page clone showcasing CSS styling and layout techniques.

**Features:**
- Custom navigation bar
- Centered search interface with interactive elements
- Favorite links section with circular icons
- Tab container with hover effects
- Responsive flexbox layouts
- Box shadows and modern UI styling

**Files:** 
- `CSS-Assignment/index.html`
- `CSS-Assignment/styles.css`

## Technologies
- HTML5
- CSS3 (Flexbox, hover effects, box-shadow)

## Getting Started
Open `index.html` in either assignment folder with a web browser to view the pages.
